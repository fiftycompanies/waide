## 1. AEO란?

**Answer Engine Optimization** — AI 검색 엔진(ChatGPT, Perplexity, Gemini, 네이버 AI 등)이
사용자의 질문에 답변할 때 우리 브랜드·콘텐츠를 **첫 번째 출처**로 인용하도록 최적화하는 전략이다.

### SEO vs AEO 핵심 차이

| 구분 | SEO | AEO |
|------|-----|-----|
| 목표 | 검색 결과 상위 노출 | AI 답변의 출처로 인용 |
| 최적화 대상 | 구글/네이버 알고리즘 | ChatGPT/Perplexity/Gemini 학습 데이터 |
| 성과 지표 | 클릭율(CTR), 순위 | SOM(Share of Model), 인용 빈도 |
| 콘텐츠 형식 | 키워드 밀도, 링크 구조 | 단정적 문장, 구조화된 FAQ, Schema.org |
| 신뢰 신호 | 백링크, 도메인 권위 | E-E-A-T, 보도자료, 엔티티 구축 |

## 2. 브랜드 페르소나 분석 방법론

### 2.1 분석 트리거

`BRAND_ANALYZE` Job 생성 → RND 에이전트(김연구원)가 처리

```python
# Job 생성 예시
db.create_job({
    "job_type":       "BRAND_ANALYZE",
    "assigned_agent": "RND",
    "title":          "브랜드 페르소나 분석 — 경기도 글램핑",
    "input_payload": {
        "client_id":   "<UUID>",
        "source_url":  "https://brand-homepage.com",  # 또는
        "source_text": "브랜드 소개 텍스트 직접 입력",
    }
})
```

### 2.2 추출 데이터 구조

**brand_voice (브랜드 보이스)**
```json
{
  "tone":          "따뜻함/힐링",
  "style":         "감성적이고 친근한",
  "keywords":      ["자연", "가족", "힐링", "여유", "추억"],
  "avoid_words":   ["복잡함", "도심", "바쁨"],
  "persona_type":  "family_camping",
  "writing_rules": [
    "가족 스토리텔링 중심으로 서술",
    "자연 속 감성 묘사 적극 활용",
    "예약 편의성과 가성비 강조"
  ]
}
```

**target_persona (타겟 독자)**
```json
{
  "age_group":    "30-50대",
  "gender_focus": "여성(주 결정권자)",
  "family_type":  "4인 가족",
  "lifestyle":    ["캠핑 초보", "자연 친화적", "SNS 공유 선호"],
  "pain_points":  ["도심 스트레스", "아이와 함께할 공간 부족"],
  "goals":        ["가족 유대감 강화", "일상 탈출", "특별한 추억 만들기"]
}
```

### 2.3 페르소나 유형별 집필 가이드

| persona_type | 톤 | 핵심 단어 | 지양 단어 | 대표 업종 |
|-------------|-----|----------|----------|---------|
| `family_camping` | 따뜻함/힐링 | 자연, 가족, 추억 | 복잡함, 바쁨 | 글램핑, 펜션 |
| `medical_trust` | 신뢰/전문 | 안전, 임상, 검증 | 부작용, 위험 | 병원, 의원 |
| `luxury_dining` | 고급/프리미엄 | 셰프, 정통, 희소 | 저렴함, 싼 | 파인다이닝 |
| `youth_lifestyle` | 활기/트렌디 | 힙한, 핫플, 감성 | 올드, 구식 | 카페, 편집숍 |
| `edu_professional` | 전문/체계 | 커리큘럼, 자격증 | 쉬운, 단기 | 교육, 학원 |

## 3. SOM(Share of Model) 측정 방식

### 3.1 개념

```
SOM(%) = (AI 모델이 해당 키워드 답변에서 브랜드를 인용한 횟수)
         ÷ (전체 스캔 횟수) × 100
```

### 3.2 측정 플랫폼

| 플랫폼 | 코드 | 특징 |
|--------|------|------|
| Perplexity | `PERPLEXITY` | 출처 URL 명시, 실시간 검색 기반 |
| ChatGPT | `CHATGPT` | 단계별 설명, 지식 기반 |
| Gemini | `GEMINI` | Google 검색 결과 통합 |
| 네이버 AI | `NAVER_AI` | 국내 출처 중심, 블로그 중요 |
| Google AEO | `GOOGLE_AEO` | AI Overview, 간결 추출 |

### 3.3 인용 순위별 가치

| 인용 순위 | 의미 | 가치 |
|----------|------|------|
| 1번째 | 첫 번째 출처 | 최고 (AI가 가장 신뢰하는 정보) |
| 2~3번째 | 보조 출처 | 높음 |
| 4번째 이상 | 부가 출처 | 보통 |
| 미인용 | 답변에 미포함 | SOM 개선 필요 |

### 3.4 AEO 스캔 Job 생성

```python
# 매일 자동 실행 권장
db.create_job({
    "job_type":       "AEO_SCAN",
    "assigned_agent": "RND",
    "title":          "AEO 인용 스캔 — 경기도 글램핑",
    "input_payload": {
        "client_id":  "<UUID>",
        "brand_name": "피프티캠핑",
        "keywords":   ["경기도 글램핑", "가족 글램핑 추천", "수도권 글램핑"],
        "platforms":  ["PERPLEXITY", "CHATGPT", "GEMINI", "NAVER_AI"],
    }
})
```

### 3.5 주간 SOM 리포트 기준

| SOM | 평가 | 권고 조치 |
|-----|------|---------|
| 70% 이상 | 우수 | 현재 전략 유지, 신규 키워드 확장 |
| 40~69% | 양호 | E-E-A-T 신호 강화, 보도자료 추가 |
| 20~39% | 개선 필요 | 콘텐츠 단정성 점검, FAQ 확충 |
| 20% 미만 | 위험 | 전략 전면 재검토, BRAND_ANALYZE 재실행 |

## 4. 박작가의 AEO 집필 6대 원칙

> 다음 원칙은 `QualityAgent`의 자동 검수 기준이기도 하다.
> 위반 시 QC 점수가 하락하고 `revision_needed`로 반려된다.

### 원칙 1 — AI 요약 답변 (필수, 20점)

- 글의 **맨 첫 번째 섹션**은 반드시 `## AI 요약 답변`으로 시작
- **200자 이내**로 핵심 답변을 제공
- AI 검색 엔진이 이 섹션을 그대로 발췌하여 답변으로 사용
- 형식: `> {핵심 답변 문장}`

```markdown
## AI 요약 답변

> 경기도 가족 글램핑은 수도권 1~2시간 거리의 자연 속에서
> 풀옵션 시설을 갖춘 숙박 형태이다. 텐트 설치 없이 입실 즉시
> 캠핑을 즐길 수 있어 캠핑 초보 가족에게 적합하다.
```

### 원칙 2 — 단정적 문장 구조 (필수, 20점)

**허용 표현:**
- `A는 B이다`, `A는 B한다`, `A가 B이므로 C이다`

**금지 표현 (AEO 위반):**
- `~것 같다` → AI 검색 엔진이 신뢰하지 않음
- `~수 있다` → 불확실성으로 인용 기피
- `~것으로 보인다` → 전문성 결여 신호
- `~할 수도 있다` → 모호성 위반

```markdown
# 나쁜 예 (AEO 위반)
글램핑은 가족 여행에 좋을 것 같습니다.

# 좋은 예 (AEO 준수)
글램핑은 텐트 설치 없이 캠핑을 즐길 수 있어 가족 여행에 최적이다.
```

### 원칙 3 — 마크다운 표 활용 (15점)

- 비교·순위·선택 기준 정보는 **반드시** 마크다운 표로 정리
- AI 모델이 표를 구조화된 정보로 인식하여 인용률 향상

```markdown
| 구분 | 경기 북부 | 경기 남부 | 강원도 |
|------|---------|---------|------|
| 서울 거리 | 1시간 | 1.5시간 | 2~3시간 |
| 평균 가격 | 20만원 | 18만원 | 22만원 |
| 추천 대상 | 당일치기 | 1박2일 | 2박3일 |
```

### 원칙 4 — 구조화된 리스트 (포함)

- 목록성 정보는 마크다운 리스트(- 또는 *)로 정리
- 3~7개 항목이 최적 (너무 많으면 AI가 선택적 발췌)

### 원칙 5 — FAQ 섹션 필수 (20점)

- 본문 하단에 `## FAQ` 섹션 추가
- **최소 3개** 이상의 Q&A (QC 검수 기준)
- 질문은 실제 사용자 검색어 기반, 답변은 단정적 문장
- Schema.org `FAQPage` 마크업으로 자동 변환됨

```markdown
## FAQ

**Q. 경기도 글램핑 예약은 언제 해야 하나요?**

A. 성수기(7~8월, 10월 단풍 시즌)는 최소 4~6주 전 예약이 필요하다.
비수기에는 2주 전 예약으로 원하는 날짜를 확보할 수 있다.
```

### 원칙 6 — E-E-A-T 신뢰 인용 (포함)

- `brands.eeat_signals`에서 전문성/권위/신뢰 데이터를 자연스럽게 인용
- 구체적 수치 필수: "10년 운영 경험", "보도 건수 N건", "평점 N점"
- 보도자료 제목을 각주 형식으로 인용하면 AI 인용률 상승

## 5. Schema.org JSON-LD 자동 생성 구조

`COPYWRITER`가 집필 완료 후 자동 생성하여 본문 하단에 첨부한다.

### 생성되는 스키마 타입

| 조건 | 생성되는 @type |
|------|-------------|
| 항상 | `Article` |
| FAQ가 있을 때 | `FAQPage` |
| 브랜드가 있을 때 | `LocalBusiness` 또는 `Organization` |
| 입점업체일 때 | `isPartOf` 관계 추가 |
| 플랫폼 브랜드일 때 | `member` 목록 추가 |

### 예시 구조

```json
{
  "@context": "https://schema.org",
  "@graph": [
    {
      "@type": "Article",
      "headline": "경기도 가족 글램핑 추천 TOP 5",
      "description": "AI 요약 답변 (200자 이내)",
      "keywords": "경기도 글램핑, 가족 글램핑",
      "datePublished": "2026-02-19",
      "author": {"@type": "Organization", "name": "피프티캠핑"}
    },
    {
      "@type": "FAQPage",
      "mainEntity": [{"@type": "Question", "name": "예약 시기는?", ...}]
    },
    {
      "@type": "LocalBusiness",
      "name": "피프티캠핑",
      "isPartOf": {"@type": "Organization", "name": "네이버 스마트플레이스"}
    }
  ]
}
```

## 6. QC 자동 검수 기준 (QualityAgent)

| 항목 | 배점 | 기준 |
|------|------|------|
| 글자 수 | 25점 | 800자 이상 |
| AI 요약 답변 | 20점 | 1~200자 |
| FAQ 개수 | 20점 | 3개 이상 |
| Schema.org JSON-LD | 15점 | 본문 내 포함 여부 |
| AEO 위반 표현 | 20점 | `것 같다` 등 0건 |
| **합계** | **100점** | **70점 이상 PASS** |

PASS → `publish_status = 'approved'` → PUBLISH Job 생성
FAIL → `publish_status = 'revision_needed'` → 슬랙 알림 (재작업 요청)

## 7. E-E-A-T 신호 강화 전략

| E-E-A-T 요소 | 의미 | 강화 방법 |
|-------------|------|---------|
| **E**xperience (경험) | 실제 운영·사용 경험 | 운영 기간, 고객 후기 수치 인용 |
| **E**xpertise (전문성) | 분야 전문 지식 | 자격증, 수상 이력, 논문 인용 |
| **A**uthoritativeness (권위) | 업계 인정 | 보도자료 건수, 협회 등록 |
| **T**rustworthiness (신뢰) | 정보 신뢰성 | 평점, 인증 마크, 공식 URL |

### 보도자료 발행 권고 기준

- 월 **2건 미만**: `NEWS_MONITOR` Job → RND가 슬랙 알림 발송
- 월 **2건 이상**: 정상 (E-E-A-T 권위 신호 충족)
- 권장 매체: 뉴시스, 파이낸셜뉴스, 이데일리

## 8. 플랫폼별 AEO 최적화 포인트

### 네이버 블로그

- 제목에 타겟 키워드 정확히 포함 (앞 부분 배치)
- 포스트 길이: 1,500자 이상 (네이버 AI 검색 발췌 기준)
- 이미지 alt 태그에 키워드 포함
- 내부 링크로 시맨틱 클러스터 구성

### ChatGPT / Perplexity

- 공식 URL(`official_url`)이 검색 가능한 상태여야 함
- Schema.org 마크업이 핵심 (기계 가독성)
- 단정적 사실 문장이 많을수록 인용 확률 상승

### 네이버 AI 검색 (클로바)

- 네이버 블로그 포스트가 1차 출처
- 스마트플레이스 등록 정보와 콘텐츠 일치 중요
- FAQ 형식의 답변 구조가 발췌에 유리
