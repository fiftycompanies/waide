Vercel API 엔드포인트 REST API

POST 프로젝트 생성

/v10/projects

Vercel에 새 프로젝트를 생성하고 Git 레포지토리를 연결합니다.

POST 환경변수 설정

/v10/projects/{id}/env

SUPABASE_URL, PROJECT_ID 등 환경변수를 자동 주입합니다.

POST 배포 트리거

/v13/deployments

소스 코드를 업로드하고 빌드/배포를 트리거합니다.

POST 도메인 연결

/v10/projects/{id}/domains

{subdomain}.waide.kr 도메인을 프로젝트에 연결합니다.

GET 배포 상태 조회

/v13/deployments/{id}

배포 진행 상태를 실시간으로 조회합니다.

POST 프로모트 (라이브 전환)

/v10/projects/{id}/promote

프리뷰 배포를 프로덕션으로 프로모트합니다.
