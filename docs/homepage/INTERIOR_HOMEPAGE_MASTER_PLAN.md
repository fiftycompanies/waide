### 1-2. AEO (Answer Engine Optimization) 키워드

AI 답변 엔진(Perplexity, Claude, ChatGPT, 네이버 AI검색)에 최적화된 질문형 키워드:

| 질문 유형 | 예시 | 답변 포맷 |
|-----------|------|-----------|
| **비용형** | "30평 인테리어 비용 얼마인가요?" | 가격대 범위 + 항목별 세부 |
| **비교형** | "강화마루 vs 강마루 뭐가 나아요?" | 비교표 + 상황별 추천 |
| **방법형** | "인테리어 시작 전 뭐부터 해야 하나요?" | 순서 리스트 (1,2,3...) |
| **추천형** | "신혼집 인테리어 어떻게 하면 좋을까요?" | 스타일 추천 + 예산 안내 |
| **질문형** | "인테리어 업체 어떻게 고르나요?" | 체크리스트 5-7개 기준 |

## 2. 블로그 대량발행 시스템

### 2-1. 블로그 유형 분류

#### A. 정보성 블로그

| 콘텐츠 포맷 | 키워드 패턴 | 글 구조 | SEO 효과 |
|-------------|------------|---------|----------|
| **비용 분석** | "~비용 총정리" | 제목→개요→항목별 비용표→절약 팁→CTA | 높음 |
| **가이드** | "~하는 방법", "~가이드" | 제목→문제 제기→단계별 설명→체크리스트→CTA | 높음 |
| **트렌드** | "2026 ~트렌드" | 제목→개요→트렌드 1~5→적용 팁→CTA | 중간 |
| **비교** | "A vs B" | 제목→개요→비교표→상황별 추천→결론→CTA | 중간 |
| **팁** | "~팁 N가지" | 제목→개요→팁 리스트→주의사항→CTA | 중간 |
| **주의사항** | "~주의사항" | 제목→핵심 요약→상세 설명→체크리스트→CTA | 낮음 |

#### B. 후기성 블로그

| 콘텐츠 포맷 | 키워드 패턴 | 글 구조 | SEO 효과 |
|-------------|------------|---------|----------|
| **시공 후기** | "{지역} {평수} 인테리어 후기" | 기본정보→Before→시공과정→After→비용공개→평가 | 매우 높음 |

### 2-3. 대량발행 전략

#### 월간 발행 계획 (업체당 8회)

| 주 | 유형 | 콘텐츠 | 키워드 전략 |
|----|------|--------|------------|
| 1주-① | 정보성 | 비용/가이드 글 | "{지역} {공간} 인테리어 비용" |
| 1주-② | 후기성 | 시공 후기 | "{지역} {평수}평 인테리어 후기" |
| 2주-① | 정보성 | 팁/트렌드 글 | "{키워드} 팁/트렌드" |
| 2주-② | 후기성 | 견적 후기 | "{지역} 인테리어 견적 공개" |
| 3주-① | 정보성 | 비교/Q&A (AEO) | "A vs B", "~어떻게?" |
| 3주-② | 후기성 | 업체 후기 | "{업체명} 인테리어 후기" |
| 4주-① | 정보성 | 주의사항/체크리스트 | "인테리어 ~주의사항" |
| 4주-② | 후기성 | 완성 포토 후기 | "{스타일} 인테리어 완성" |

#### 대량발행 파이프라인
```
[키워드 매트릭스] → [템플릿 선택] → [AI 초안 생성] → [업체 데이터 삽입] → [사진 삽입] → [SEO 메타 자동생성] → [JSON-LD 스키마 삽입] → [발행]
```

#### 월 50개 업체 기준 = 400개 글/월
- 정보성: 200개 (업체별 4개 × 50개)
- 후기성: 200개 (업체별 4개 × 50개)
- 지역 키워드 커버리지: 50개 지역 × 8개 키워드 = 400개 조합

```typescript
// app/sitemap.ts
export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const company = await getCompanyConfig();
  const portfolios = await getPortfolios(company.id);
  const blogPosts = await getBlogPosts(company.id);

  const staticPages = [
    { url: baseUrl, lastModified: new Date(), changeFrequency: 'weekly', priority: 1.0 },
    { url: `${baseUrl}/portfolio`, priority: 0.9 },
    { url: `${baseUrl}/services`, priority: 0.8 },
    { url: `${baseUrl}/blog`, priority: 0.8 },
    { url: `${baseUrl}/reviews`, priority: 0.7 },
    { url: `${baseUrl}/contact`, priority: 0.7 },
    { url: `${baseUrl}/faq`, priority: 0.6 },
    { url: `${baseUrl}/about`, priority: 0.5 },
    { url: `${baseUrl}/location`, priority: 0.5 },
  ];

  const portfolioPages = portfolios.map(p => ({
    url: `${baseUrl}/portfolio/${p.slug}`,
    lastModified: p.updated_at,
    priority: 0.7,
  }));

  const blogPages = blogPosts.map(p => ({
    url: `${baseUrl}/blog/${p.slug}`,
    lastModified: p.published_at,
    priority: 0.6,
  }));

  return [...staticPages, ...portfolioPages, ...blogPages];
}
```
